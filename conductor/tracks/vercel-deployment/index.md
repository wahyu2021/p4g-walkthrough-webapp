# Vercel Deployment

## Deskripsi
Track ini bertujuan untuk mendeploy aplikasi P4G Walkthrough secara penuh (Frontend Vite React & Backend Express MongoDB) ke platform hosting Vercel. Karena Vercel adalah platform Serverless, kita perlu melakukan beberapa penyesuaian pada arsitektur Backend Express agar dapat berjalan optimal sebagai Serverless Function, serta mengonfigurasi Monorepo agar Vercel dapat mem-build dan menjalankan kedua service tersebut.
