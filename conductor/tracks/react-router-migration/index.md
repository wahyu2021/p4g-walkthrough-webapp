# React Router Migration

**Status:** Planned
**Priority:** High

## Description
Migrate the application's navigation from a state-based conditional rendering approach (`currentView` in `App.tsx`) to standard URL-based routing using `react-router-dom`. This will enable bookmarking, deep linking, and proper browser history (Back/Forward buttons) which is crucial for a wiki-style application.
