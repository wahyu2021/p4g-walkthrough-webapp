# Specification: Interactive Dungeon Guide (The Midnight Hub)

## 1. Objective
Transform the static dungeon list into an interactive strategy hub where players can look up enemy weaknesses, read boss tactics, and track floor progress.

## 2. Key Features
- **Midnight Bestiary**:
  - A toggle inside each `DungeonCard` to show/hide the list of enemies.
  - A visual grid of enemies with colored icons for weaknesses (e.g., Red flame icon with "Wk").
  - Search/Filter within the bestiary.
- **Boss Strategy View**:
  - "Boss Info" button that opens a detailed card with Shadow stats and the "Advice" text extracted from the guide.
- **Loot Navigator**:
  - List of all items found in Regular and Rare chests.
  - "Shuffle Time" Persona list to help players know which Personas to collect.
- **P4G Aesthetic**:
  - Use TV scanline effects and glitch animations for the transition into dungeon details.

## 3. Success Criteria
- Users can click on a dungeon and see a detailed breakdown of its inhabitants and loot.
- Weakness information is visually clear and glanceable (critical for fast gameplay).
- Mobile-friendly tables for enemy stats.
