# Velvet Room: Fusion Roadmap

**Status:** Planned
**Priority:** High

## Description
A comprehensive, level-gated guide for Persona fusions. This feature provides step-by-step recipes for creating the most optimal Personas from the very beginning of the game (Early Game) through to the Secret Bosses (End Game). It acts as a roadmap, so players never feel lost regarding which Persona they should be trying to fuse at their current level.
