# Deep Linking & Dungeon Revamp

**Status:** Planned
**Priority:** High

## Description
This track focuses on deepening the React Router integration. It will change the Walkthrough timeline to update the URL when switching months (e.g., `/walkthrough/april`) and completely revamp the Dungeons page. The Dungeons page will shift from a grid of collapsible cards to a single, fully-expanded detail view with its own navigation selector, utilizing URL slugs (e.g., `/dungeons/yukiko`).
